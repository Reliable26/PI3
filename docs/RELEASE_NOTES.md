# Release Notes

## v0.9.19 Alpha - Ownership Validation Visibility

- Adds a visible Ownership Validation panel so source validation is no longer hidden in QA or details tabs.
- Shows official ownership source reachability.
- Shows how many Property Intelligence Records currently have owner data from permit/GIS fields.
- Shows how many official ownership verification links are attached.
- Clarifies that ownership-change opportunities are not created until Register of Deeds automation is certified.
- No new branding. No UI redesign.


## v0.9.19 Alpha - Ownership Drilldown

### Added
- Visible Matched Ownership Records drilldown under Ownership Validation.
- QA links from ownership metrics to the drilldown table.
- Owner, parcel, source, and official verification links are now visible in one place.
- Ownership summary appears inside expanded property Overview when owner data exists.
- Ownership verification links also appear inside the Details tab.

### Clarified
- Ownership matched means owner data was found from permit/GIS/ownership validation sources.
- Ownership change opportunities are still pending Register of Deeds automation certification.
