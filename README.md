# Commercial Property Intelligence v0.9.6 Alpha

Pilot workflow build focused on master/detail layout behavior and source-aware permit evidence links.

## Install
Copy the contents of this package into the PI3 repository root, commit, push, and run GitHub Actions -> Update Intelligence.

## Highlights
- Removes inactive hamburger menu.
- Keeps desktop workspace from stacking too early.
- Keeps property list, map, and Property Intelligence Record visible together on wider screens.
- Adds source-aware permit actions:
  - Copy Permit #
  - Open the correct official permit search page for the source system
  - View County Source Record
- Adds source labels to permit timeline records so users know where each permit came from.
