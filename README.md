# Reliable Intel v0.9.3 Alpha

UI/UX pilot build focused on making the existing intelligence easier to use during the 30-day pilot.

## Install
1. Copy all files into the PI3 repo root.
2. Commit: `Reliable Intel v0.9.3 alpha intelligence workspace UI`
3. Push.
4. Run GitHub Actions -> Update Intelligence -> Run workflow.

## Validate
- Dashboard should keep the same data pipeline and connectors from v0.9.2.
- Today's Opportunities should support search, sort, category filter, and minimum score filter.
- Property timelines should display newest activity first.
- Source Health and Intelligence QA should be collapsed by default.
- The layout should feel closer to an executive intelligence workspace instead of a raw opportunity feed.
