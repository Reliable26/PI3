# PI v0.2.3 - Permit Intelligence Preview

This release adds the first non-fire opportunity class: Capital Improvement / Permit Intelligence.

Install by copying the contents of this package into the root of the PI3 repository, replacing existing files, committing, pushing, and running GitHub Actions > Update Intelligence.

Commit message:

```text
PI v0.2.3 Permit Intelligence Preview
```

This release still uses public RSS/news signals for permit opportunities and validates public permit source connectivity. Direct Mecklenburg/Charlotte permit record extraction is the next source-specific parser step.
